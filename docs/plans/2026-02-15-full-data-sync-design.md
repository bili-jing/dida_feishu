# 全量数据同步到飞书多维表格 - 设计文档

## 目标
将苏沐林账户的所有滴答清单数据（含 2,862 个附件）完整同步到飞书多维表格，支持图片预览、文件在线查看、标签筛选和清单分组。

## 数据概览
- 4,784 条任务，21 个项目
- 2,862 个附件（1,913 图片、732 PDF、98 DOC、72 XLS、31 PPT、6 视频等）
- 总附件大小：~3.2 GB，无超过 20MB 的文件
- 4 个标签（微信采集 1525 条为主）
- 74 个含子任务的任务

## 新字段结构

| 字段名 | 类型 | 变化 |
|--------|------|------|
| 任务名称 | 文本(1) | 不变 |
| 状态 | 单选(3) | 不变 |
| 优先级 | 单选(3) | 不变 |
| 所属清单 | 单选(3) | 从文本改为单选 |
| 内容 | 文本(1) | 清洗掉附件标记 |
| 附件 | 附件(17) | 新增 |
| 子任务 | 文本(1) | 新增 |
| 标签 | 多选(4) | 不变 |
| 创建时间 | 日期(5) | 不变 |
| 截止时间 | 日期(5) | 不变 |
| 完成时间 | 日期(5) | 不变 |
| 滴答ID | 文本(1) | 不变 |

## 附件流程
1. 从 SQLite raw JSON 解析 attachments 数组
2. 从 Dida API 下载：`https://api.dida365.com{path}` + Cookie `t=<token>`
3. 本地缓存到 `./downloads/<attachment_id>/<filename>`
4. 上传飞书：`client.drive.media.uploadAll()`, parent_type = bitable_image/bitable_file
5. 缓存 file_token 到 attachment_cache 表避免重复上传
