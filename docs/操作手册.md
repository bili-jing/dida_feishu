# 滴答清单同步工具 — 操作手册

本工具可以将你在滴答清单中的任务数据同步到飞书多维表格中查看和管理。

---

## 一、启动程序

### Windows

双击 `dida-feishu.exe` 即可启动。

### macOS

双击 `启动.sh` 文件即可启动。

> **首次运行注意事项：**
>
> 1. 右键点击 `启动.sh` → 选择「打开方式」→「终端」
> 2. 如果弹出安全提示，点击「打开」
> 3. 首次运行后，后续双击即可直接打开
>
> 如果程序被系统杀掉（提示 `killed`），打开「终端」执行以下命令后重试：
> ```
> xattr -cr 程序所在文件夹路径/dida-feishu-macos
> ```
> 启动脚本已内置此处理，正常使用 `启动.sh` 即可自动解决。

启动后会看到主菜单界面，使用键盘 **上下箭头** 选择选项，按 **回车** 确认。任何时候按 **Ctrl+C** 可以退出程序。

---

## 二、主菜单

```text
◆  请选择用户
│  ● 张三 (微信, 28天有效)
│  ○ 李四 (密码, 15天有效)
│  ○ 添加新用户
│  ○ 删除用户
```

主菜单显示所有已登录的用户。每个用户后面会标注登录方式和剩余有效天数。

### 选择已有用户

直接选中用户名，按回车进入该用户的操作菜单。

### 添加新用户

选择「添加新用户」后，会提示选择登录方式：

#### 手机号/邮箱登录

1. 选择「手机号/邮箱登录」
2. 输入滴答清单账号（手机号或邮箱）
3. 输入密码
4. 可选输入一个备注名称（方便区分多个账号）

#### 微信扫码登录

1. 选择「微信扫码登录」
2. 可选输入备注名称
3. 屏幕上会显示一个二维码（Windows 会自动弹出图片）
4. 用微信扫描二维码并确认授权
5. 等待程序提示「微信授权成功」

> 登录成功后，程序会自动保存登录凭据，有效期约 30 天。过期后重新操作即可自动刷新。

### 删除用户

选择「删除用户」，然后选择要删除的账号，确认后会清除该用户的所有本地数据。

---

## 三、用户操作菜单

选择用户后进入操作菜单：

```text
◆  张三 - 操作
│  ● 拉取最新数据 (本地 156 条)
│  ○ 数据概况
│  ○ 搜索任务
│  ○ 待同步任务 (新增 3 + 修改 2)
│  ○ 同步到飞书 (5 条待更新)
│  ○ 飞书配置 (已绑定)
│  ○ AI配置 (已配置)
│  ○ 文档管理
│  ○ ← 返回主菜单
```

以下逐一说明每个功能。

---

### 3.1 拉取最新数据

从滴答清单服务器获取你的最新任务数据，保存到本地。

**拉取内容包括：**

- 所有项目/清单
- 进行中的任务
- 已完成的任务
- 已放弃的任务
- 垃圾桶中的任务

**拉取完成后会显示数据概况：**

```text
◇  数据概况 ──────────────────────────────────╮
│  项目: 5  未完成: 23  已完成: 120  已放弃: 3  总计: 146  │
│  新增: +5 条                                              │
├──────────────────────────────────────────────╯
```

> 每次同步到飞书之前，建议先「拉取最新数据」，确保本地数据是最新的。

---

### 3.2 数据概况

查看当前用户的数据统计和飞书同步状态：

```text
◇  数据概况 ──────────────────────────────────╮
│  项目: 5  未完成: 23  已完成: 120  已放弃: 3  总计: 146  │
│                                                          │
│  飞书同步状态:                                            │
│    已同步: 141  未同步: 3  已修改: 2  未变: 139           │
│    链接: https://xxx.feishu.cn/base/xxx                  │
├──────────────────────────────────────────────╯
```

其中：

- **已同步**：已经推送到飞书的任务数量
- **未同步**：新增的、还没推送到飞书的任务
- **已修改**：在滴答清单中修改过、但飞书还是旧数据的任务
- **未变**：两边数据一致的任务

---

### 3.3 搜索任务

按关键词搜索本地保存的任务：

1. 输入搜索关键词（支持标题和内容匹配）
2. 从搜索结果中选择一条查看详情
3. 直接按回车（不输入关键词）返回上级菜单

搜索结果会标注同步状态：

- **✓** 已同步到飞书
- **○** 未同步

---

### 3.4 待同步任务

> 此选项仅在有未同步或已修改的任务时出现。

查看哪些任务还没有推送到飞书，或者修改后还没更新：

- **未同步的任务**：新拉取的、还没推送到飞书的
- **已修改的任务**：在滴答清单修改过，飞书端还是旧数据的

选择具体任务可查看详情。

---

### 3.5 同步到飞书

将本地任务数据推送到飞书多维表格。

> 首次使用需要先绑定飞书（见 3.6 飞书配置）。

#### 首次同步

如果从未同步过，会提示：

```text
│  尚未同步到飞书，将执行首次全量同步
◆  开始全量同步？
│  ● 是 / ○ 否
```

确认后程序会：

1. 在飞书中创建一个新的多维表格
2. 自动创建「全部任务」「进行中」「已完成」「看板」四个视图
3. 将所有任务写入表格
4. 同步完成后显示飞书链接，点击即可打开查看

#### 后续同步

已经同步过的用户，会显示当前同步状态：

```text
◇  同步状态 ───────────────────╮
│  飞书链接: https://xxx...     │
│  已同步: 141  新增: 3  修改: 2│
├──────────────────────────────╯
◆  选择同步方式
│  ● 增量同步 (更新 5 条)
│  ○ AI 增量更新
│  ○ 全量重建
│  ○ ← 返回
```

- **增量同步**：只更新有变化的任务（推荐，速度快）
- **AI 增量更新**：只更新 AI 生成的标题、摘要、链接内容，不动附件（需先配置 AI，见 3.7）
- **全量重建**：创建一个全新的飞书表格，重新写入所有数据

> 「AI 增量更新」选项仅在配置了 AI 后才会出现。

选择**增量同步**后，会询问是否跳过附件：

```text
◆  是否跳过附件？
│  ● 否 / ○ 是
```

- 选「否」：会下载滴答中的图片、文件等附件并上传到飞书（较慢，但数据完整）
- 选「是」：跳过附件，只同步文字内容（较快）

> 附件处理有缓存，同一个文件不会重复下载和上传。

#### 全量重建时处理旧表格

选择「全量重建」后，如果已经有同步过的表格，程序会提示你如何处理旧表格：

```text
◆  检测到旧表格，如何处理？
│  ● 删除旧表格后重建 (旧表格移入飞书回收站)
│  ○ 保留旧表格，创建新的
│  ○ 取消
```

- **删除旧表格后重建**：先删除飞书中的旧表格（移入回收站，30 天内可恢复），然后创建新表格
- **保留旧表格，创建新的**：旧表格保留不动，另外创建一个新表格
- **取消**：放弃本次全量重建

> 旧表格删除后会进入飞书回收站，30 天内可以在回收站中恢复。

---

### 3.6 飞书配置

管理飞书应用凭据。同步到飞书需要先绑定飞书应用。

#### 首次绑定

1. 选择「飞书配置」
2. 按照提示输入飞书 App ID 和 App Secret
3. 程序会自动验证凭据是否有效
4. 验证成功后保存

> **如何获取 App ID 和 App Secret？**
>
> 1. 打开 [飞书开放平台](https://open.feishu.cn/app)
> 2. 创建一个企业自建应用
> 3. 在应用详情页找到 App ID 和 App Secret
> 4. 在「权限管理」中开通「多维表格」和「云空间」权限
> 5. 创建应用版本并发布

#### 已绑定后

```text
◇  当前飞书配置 ──────────╮
│  App ID: cli_a9...bca    │
│  当前文档: https://...   │
│  历史文档: 3 个          │
├─────────────────────────╯
◆  飞书配置操作
│  ● 管理文档
│  ○ 换绑
│  ○ 解绑
│  ○ ← 返回
```

- **管理文档**：进入文档管理，查看和删除飞书中的历史表格（详见 3.8）
- **换绑**：更换为其他飞书应用的凭据
- **解绑**：移除凭据，解绑后需重新绑定才能同步

> 「历史文档」数量只在有历史文档时显示。

---

### 3.7 AI配置

配置 AI 功能后，同步时可以自动为任务生成标题摘要、分析链接内容等。

> AI 功能是可选的，不配置也不影响正常同步。

#### 首次配置

1. 选择「AI配置」
2. 按提示输入火山引擎 API Key
3. 输入文本模型名称（默认 `doubao-seed-2-0-lite-260215`）
4. 可选配置视觉模型（用于图片理解）
5. 程序会自动验证配置是否有效

> **如何获取 API Key？**
>
> 打开 [火山引擎控制台](https://console.volcengine.com/ark) ，创建 API Key 并开通对应模型。

#### 已配置后

```text
◇  当前 AI 配置 ──────────╮
│  模型: doubao-seed-2-0   │
├─────────────────────────╯
◆  AI 配置操作
│  ● 重新配置
│  ○ 移除配置
│  ○ 清除缓存
│  ○ ← 返回
```

- **重新配置**：更换 API Key 和模型
- **移除配置**：关闭 AI 功能，同时清除缓存
- **清除缓存**：保留配置，但清除已生成的 AI 摘要，下次同步时会重新生成

配置 AI 后，同步菜单中会出现「AI 增量更新」选项（见 3.5）。

---

### 3.8 文档管理

管理飞书云空间中由本工具创建的多维表格。每次「全量重建」都会创建一个新表格，旧表格不会自动删除。通过文档管理可以查看和清理这些历史表格。

**进入方式：**

- 用户操作菜单 →「文档管理」
- 飞书配置菜单 →「管理文档」

进入后，程序会扫描飞书云空间，列出所有包含「滴答清单」的多维表格：

```text
◆  选择文档
│  ● ★ 滴答清单 - 张三  (当前使用)  2024/2/20
│  ○   滴答清单 - 张三              2024/2/15
│  ○   滴答清单 - 张三              2024/2/10
│  ○ 批量删除非当前文档
│  ○ ← 返回
```

- **★** 标记的是当前正在使用的表格
- 选择单个历史文档后，可以查看链接或删除
- **批量删除非当前文档**：一键删除所有非当前使用的历史表格

> 删除的表格会进入飞书回收站，30 天内可以恢复。当前使用的表格不会被删除。

## 四、同步后的飞书表格

同步完成后，在飞书中打开链接，你会看到一个多维表格，包含以下内容：

### 表格字段

| 列名 | 内容 |
|------|------|
| **内容标题** | 任务的标题 |
| **类型** | 自动识别的任务类型（纯文本、聊天记录、图片、文件、链接等） |
| **链接** | 任务中包含的网址（可点击打开） |
| **内容** | 任务的正文内容 |
| **附件** | 图片和文件（可点击预览） |
| **标签** | 任务的标签 |
| **子任务** | 子任务清单（☑ 已完成 / ☐ 未完成） |
| **状态** | 进行中 / 已完成 / 已放弃 |
| **所属清单** | 任务属于哪个清单/项目 |
| **优先级** | 无 / 低 / 中 / 高 |
| **创建时间** | 任务创建的时间 |
| **截止时间** | 截止日期 |
| **完成时间** | 完成的时间 |

### 预置视图

| 视图 | 说明 |
|------|------|
| **全部任务** | 显示所有任务 |
| **进行中** | 只显示进行中的任务 |
| **已完成** | 只显示已完成的任务 |
| **看板** | 按状态分列的看板视图 |

> 你也可以在飞书中自由创建新的视图、筛选、排序。

## 五、配合飞书 AI 功能

同步完成后，你可以在飞书多维表格中添加 AI 智能列，让 AI 自动处理数据：

| 你想做的事 | 添加的 AI 捷径 | 让它读取哪一列 |
|-----------|---------------|-------------|
| 总结长文本/聊天记录 | 「总结」 | 内容 |
| 分析 PDF 或图片内容 | 「Kimi 阅读助手」 | 附件 |
| 抓取网页链接内容 | 「AI 读取网页链接」 | 链接 |
| 自动给任务分类 | 「分类」或「智能打标」 | 内容 |

**添加方法：** 在表格中点击最右边的 **+** 新增列 → 在字段类型下方点击「探索字段捷径」 → 选择对应的 AI 捷径 → 配置后点击生成。

开启「自动更新」后，每次同步新数据进来，AI 会自动处理。

## 六、常见操作流程

### 日常使用（推荐流程）

```text
1. 打开程序
2. 选择你的账号
3. 选择「拉取最新数据」 → 获取滴答清单最新任务
4. 选择「同步到飞书」→「增量同步」 → 推送变化到飞书
5. 打开飞书链接查看
```

### 首次使用完整流程

```text
1. 打开程序
2. 选择「添加新用户」→ 登录滴答清单
3. 程序自动拉取数据
4. 提示是否绑定飞书 → 选「是」→ 输入 App ID 和 App Secret
5. 返回菜单，选择「同步到飞书」→ 确认全量同步
6. 等待同步完成，获得飞书链接
```

### 更换飞书应用

```text
1. 进入用户菜单
2. 选择「飞书配置」→「换绑」
3. 输入新的 App ID 和 App Secret
4. 下次同步时选择「全量重建」创建新表格
```

## 七、注意事项

1. **数据方向**：同步是从滴答清单到飞书的单向同步。在飞书中的编辑不会回写到滴答清单。

2. **飞书编辑安全性**：
   - 如果你只在飞书中做了修改（滴答没改），下次增量同步不会覆盖你的修改
   - 如果同一个任务在滴答和飞书都改了，飞书的修改会被滴答的数据覆盖

3. **登录有效期**：登录凭据约 30 天有效。过期后重新「拉取最新数据」会自动重新登录。

4. **数据安全**：所有数据保存在程序所在文件夹的 `db` 目录下，不会上传到任何第三方服务器。

5. **多用户**：支持多个滴答清单账号，每个账号可以绑定不同的飞书应用，数据互不干扰。
